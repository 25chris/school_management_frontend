# School Management App

Welcome to the repository for the **School Management** app, a comprehensive Flutter frontend design utilizing Stacked CLI.
This app is designed to manage all school activities, providing an all-in-one solution for courses, units, schedules, grades, tuition, events, teachers, students, and more.

## About the Project

The School Management app aims to streamline the administration of school activities through a user-friendly interface and robust functionality.
It is designed to cater to the needs of administrators, teachers, students, and parents, making school management efficient and effective.

### Features

- **Course Management**: Create, update, and manage courses and units.
- **Scheduling**: Organize and manage school schedules for classes and events.
- **Gradebook**: Record and track student grades and performance.
- **Tuition Management**: Handle tuition fees, payments, and billing.
- **Event Management**: Plan and manage school events and activities.
- **Teacher and Student Management**: Maintain detailed records of teachers and students.

## Technology Stack

The app is built using Flutter for a responsive and intuitive user experience, leveraging Stacked CLI for efficient state management and project structure.

### Key Technologies Used:

- **Flutter**: A UI toolkit for crafting natively compiled applications for mobile, web, and desktop from a single codebase.
- **Stacked CLI**: A tool for state management and project structure in Flutter.
- **Firebase**: Backend services for authentication, database, and cloud functions.
- **Provider**: State management solution for managing app state.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

- Flutter SDK: [Installation Guide](https://flutter.dev/docs/get-started/install)
- Dart SDK: Included with Flutter SDK
- Android Studio: [Download and Install Android Studio](https://developer.android.com/studio)
- Git: [Download and Install Git](https://git-scm.com/downloads)

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/25chris/school_management.git
   ```
2. Navigate to the project directory
   ```sh
   cd school_management
   ```
3. Install dependencies
   ```sh
   flutter pub get
   ```
4. Run the app on an emulator or physical device
   ```sh
   flutter run
   ```
